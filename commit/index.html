Ren Yuan

Commit

Video / Immersive video, Variable, 2018-Present


"Commit" is a series of visualization and sonification works created by Ren Yuan based on GitHub commit history. By analyzing developers' code commit data on GitHub, the work uses data-driven generation to generate abstract visual dynamics and sound sequences, showing from a macro level the process of developers building code projects from scratch through open source collaboration on the Internet. Ren Yuan transforms the technical act of code commit into a unique audiovisual art experience, and explores the diversity of visual and sound expressions of data through a variety of mapping algorithms and rules. As a dynamically developing series of works, "Commit" will be continuously updated with Ren Yuan's research and continue to explore more possibilities of artistic practice.

The video version of the series is on X.

The data of the work comes from GitHub's Git Commit History, including the developers' commit times, types, and collaboration relationships. Driven by data, the work translates the process of developers jointly building knowledge and technology in Internet collaboration into perceptible visuals and sounds, hoping to guide the audience to rethink the human effort and creativity behind technical collaboration.



Visualization

Each point is a developer, each curve is a commit, and the file tree of the code project grows as developers collaborate. The work also explores a lot of visual forms and spatial presentation methods, which is divided into online Internet video version and offline space exhibition version, hoping to bring a multi-dimensional experience to the audience.



Sonification

Two MIDI generation tools were developed to generate melody and rhythm, convert data into sound, give melody and rhythm to the code commit process, and trying to create a unique sound representation. The audience can perceive the melody of the data and the rhythm of the collaborative process through hearing. More synthesizers are planned in the future, hoping to further explore the various possibilities of data sonification.



Different Data-Driven

Different data and mapping algorithms are used to drive visualization and sonification systems, forming different dynamic processes of visual and sound. The collaborative process of each code project has its own mode and frequency. This difference also brings richness expressions to visual and sound, reflecting the diversity and complexity behind the data.



Immersive Space



Combination

The commit data is used to generate trigger data, and the trigger data is used to drive the generation and dynamic change of abstract visual. By combining abstract visual with visualization, it is hoped that the commit data of each code repository can be presented in a different way.




L-Shaped Screen



Data-Driven Latent Space Walk

In this series of works, many AI underlying technology code projects are visualized and sonified, and attempts are made to combine visualization with AI systems to drive AI to walk in the latent space through data. The specific method is to use the visualization system to generate conditional control images, input them into AI, and drive AI to walk in the latent space, so as to achieve the process of developers commit history data to drive AI generation. The purpose of this is to hope that while the audience is experiencing AI tools and marveling at the increasing maturity of AI technology, they can also pay attention to the significance of human beings promoting the development of scientific and technological intelligence through collective wisdom behind these technologies.
