Selected projects
W = Fs

Interactive Installation, Real-Time, 2022
Commissioned by Nike

W = Fs is an interactive installation. User movement data is obtained through sensors, and abstract graphics are generated when the user does work.

F = ma

Interactive Installation, Real-Time, 2021
Commissioned by Nike

F = ma is an interactive installation. User movement data is obtained through sensors, then converted into force control lines that form the shape of the user's body.

Car(human)

Interactive Installation, Real-Time, 2019
Commissioned by Volkswagen

Car(human) is an interactive installation. Through sensors, data on the user's operation of the steering wheel, throttle, turn signal, and windshield wiper are obtained to generate abstract visual and sound in real-time.

Chinese Zodiac

Interactive Installation, Real-Time, 2019
Commissioned by Magnetic Field Technology

Chinese Zodiac is an interactive installation. It is used at a public space entrance to display the current visitor's zodiac sign, the total number of visitors from each zodiac sign, and the total number of historical visitors.

WHY NOT

Data-Driven Animation, 1'15", 2019
Commissioned by GeekPark

WHY NOT is a data-driven animation based on code generation. The visual creation is based on brainwave sensors that collect brainwave data from specific users as they think about questions during an interview.

IF

Data-Driven Animation, 1'30", 2018
Commissioned by GeekPark

IF is a data-driven animation based on code generation. The visual creation is based on thousands of user questionnaires about artificial intelligence on the web.

Speed

Interactive Installation, Real-Time, 2017
Commissioned by Nike

Speed is an interactive installation. The user's running speed is obtained in real time through the sensor, and the abstract visual form and dynamics are controlled based on the speed data.

Swarma

Code-Generated Animation, 36", 2017
Commissioned by Swarma Club

Swarma is an abstract animation based on code generation.

Live Coding Piano

Live Performance, Real-Time, 2017
Concept, design, programming: Ren Yuan
Music: Yue Xuan

Live Coding Piano is a live performance that combines real-time programming with piano playing. Ren Yuan visualized Yue Xuan's music through live coding, presenting the entire process and result of creation to the audience.

Play()

Live Performance, Real-Time, 2016
Concept, design, programming: Ren Yuan
Music: Hong Qile

Play() is a live performance. Ren Yuan uses a program to analyze Hong Qile's music signals and control abstract visual forms and movements in real-time.

X-LIVE
